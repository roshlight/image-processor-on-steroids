Тебе дано изображение и пользовательский запрос.

Пользовательский запрос:
{user_instruction}

Доступные инструменты:
{available_tools_json}

Верни строго JSON следующего формата:

{{
  "task_type": "enhancement | add_text | geometry | filter | object_removal | multi_step_edit | unknown",
  "actions": [
    {{
      "tool": "tool_name",
      "args": {{}},
      "reason": "почему выбран этот инструмент"
    }}
  ],
  "user_message": "короткое сообщение пользователю о плане"
}}

Правила:
- Используй только инструменты из списка.
- Разбивай сложный запрос на атомарные действия.
- Сохраняй порядок действий пользователя.
- Если пользователь просит сделать фото ярче, используй adjust_brightness.
- Если пользователь просит повысить контраст, используй adjust_contrast.
- Если пользователь просит повысить насыщенность, используй adjust_saturation.
- Если пользователь просит сделать фото резче, используй sharpen.
- Если пользователь просит размыть изображение, используй gaussian_blur.
- Если пользователь просит повернуть изображение на 90 градусов вправо, используй rotate90_cw.
- Если пользователь просит повернуть изображение на 90 градусов влево, используй rotate90_ccw.
- Если пользователь просит добавить надпись, используй add_text.
- Если пользователь просит убрать объект, пока используй create_center_mask или create_bbox_mask, затем inpaint_opencv.
- Если запрос невозможно выполнить текущими инструментами, верни task_type="unknown" и actions=[].
